# computer-info-printer

## Purpose
Use Node.js to print out info about your computer system.

## System Requirements
NodeJS is required to run the Javascript file on your own machine.

You can download the current version of NodeJS here if you don't have it: https://nodejs.org/en/download/current

You can check to see if NodeJS is already installed on your system by running this command in a terminal:

```sh
node -v         # alternatively, you could also use 
node --verison  # this command 
```

Either option will print the version of NodeJS you're using to your terminal if it's been properly installed.

## External Dependencies
None so far, but NodeJS **must** be installed on your system in order to run the script
